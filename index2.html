Abstract

Satellite communication is crucial for many modern applications but is vulnerable to intentional interference such as narrowband jamming. This project proposes an automated detection pipeline using simulated BPSK signals and a Random Forest classifier trained on simple time-domain statistical features. The system achieves very high detection accuracy and can be extended to other modulations and real signal captures.

Problem Statement

Satellite links are susceptible to interference from both accidental sources and intentional jamming. Narrowband jamming, although occupying a small portion of the spectrum, can severely degrade demodulation for BPSK signals. Detecting such interference quickly allows ground stations to take mitigation steps.

Applications
  • Ground-station monitoring
  • Defense & security
  • Satellite link health diagnostics

Methodology

Signal Generation

Generate BPSK baseband signals by mapping random bits to ±1 and sampling at a moderate rate. Create both clean signals and jammed variants by adding narrowband cosine interferers with controllable amplitude and frequency. This simulates realistic jamming scenarios.

Feature Extraction

Extract simple time-domain features from each signal: mean, standard deviation, and variance. These features are sufficient to separate clean and interfered signals in this controlled simulation. Optionally, spectral features like power spectral density or higher-order statistics can be included for improved robustness.

Dataset Preparation

Build a dataset consisting of labeled clean (0) and jammed (1) signals. In this example, 200 total signals (100 clean + 100 jammed) are generated. Split into 70% training and 30% testing sets. Standardize features to improve classifier performance.

Machine Learning

Train a Random Forest classifier (100 trees) on the extracted features. Evaluate performance using accuracy and confusion matrix. This approach demonstrates a simple yet effective ML pipeline for interference detection. Future enhancements can include deep learning models using spectrogram images or real satellite signal captures.

Key Points
  • Simulate realistic BPSK signal scenarios.
  • Inject narrowband jamming for testing classifier robustness.
  • Extract simple yet effective statistical features.
  • Train/test Random Forest classifier for binary classification.
  • Visualize results using feature histograms and confusion matrix.
